Estilo do LiVES

Tipografia

Títulos usam a Source Sans Pro. É interessante usar o peso 600 na fonte (semi-bold).

Textos básicos utilizam a Open Sans

Códigos-fonte utilizam a JetBrains Mono com ligaduras ativadas == -> >>=

Dependendo da situação (como uma citação poética), uma fonte decorativa interessante é a Oleo Script Swash Caps

Em longos textos impressos, utilizar a Source Serif Pro é mais interessantes que a Open Sans

Utilize a Source Sans Pro para fórmulas lógicas e matemáticas:

λx.(x+1) * (x/100)

Cores

O fundo dos estilos do LiVES são brancos por padrão. Uma versão dark-mode será desenvolvida em breve. As cores são para elementos visuais menores, utilize sempre texto preto em fundo branco.

  • Cor primária: #00d463 (Malachite)
  • Cor secundária: #454545 (Onix)
  • Complemento para detalhes: #9f564f (Tart Orange)
  • Complemento para detalhes: #eee5e9 (Lavender Blush)
  • Complemento para detalhes: #92dce5 (Middle Blue)
  • Complemento para detalhes: #266dd3 (Celtic Blue)
  • Complemento para detalhes: #fec601 (Mikado Yellow)

Dicas Gerais

  • Priorize um visual mais limpo, minimalista, sem muitas decorações.
  • Em apresentações, escreva menos e use tamanhos de fonte maiores. O texto em si será a parte elegante.
